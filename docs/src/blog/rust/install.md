## 环境变量

:::tip 注意
`\.cargo\bin` 为相对路径，例如

```txt
C:\Users\administrator\.cargo\bin
```

:::

## 配置编译

:::tip 注意
`rust` 编译依赖 `VS`，此处我们用其他方法，也可以用安装时选择安装的 `Visual Studio`
:::

```bash
rustup toolchain install stable-x86_64-pc-windows-gnu
```

```bash
rustup default stable-x86_64-pc-windows-gnu
```
