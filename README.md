# lazy-blog

[lazy-blog](https://lazydd.github.io/lazy-blog/)

## 开始

```bash
# 1. 克隆仓库
git clone https://github.com/Lazydd/lazy-blog.git
# 2. 安装pnpm
npm install -g pnpm
# 3. 安装依赖
pnpm install
# 4. 运行
pnpm dev
# 5. 部署
pnpm build
```
